Variables and Constants
let - определение, привязывание переменной
по умолчанию все переменные при определении являются неизменяемыми (immutable)
можно сделать переменные изменяемыми добавление ключевого слова mut (mutable) 
Различия между неизменяемой переменной и константой:
1. с константами нельзя использовать mut
2. константы объявляются с помощью ключевого слова const, переменные с помощью let
3. тип хранимых в них значений в константе должен быть указан явно
4. константы могут быть объявлены в любой области видимости, включая глобальную
5. константы можно установить только в константное выражение
Затемнение (переменных)
Объявление переменной с уже существующим именем и изменение её значения
Отличие затемнения от объявления переменной с помощью mut:
1. Затемненная переменная остаётся неизменяемой
2. Затемненная переменная может быть другого типа значения, сохраняя прежнее имя

Data Type
любое значение в Rust является определённым типом данных
Типы данных делятся на простые и сложные
Простые это:
1. целые числа
2. числа с плавающей точкой
3. логический тип
4. символы

Размер Знаковый Беззнаковый
  8-bit     i8 		u8		
 16-bit		i16		u16			
 32-bit		i32		u32		
 64-bit		i64		u64			
128-bit		i128	u128
   arch		isize	usize

isize и usize в зависимости от ОС либо 32-bit либо 64-bit

Целое число может быть выраженно в любой системе счисления 
и все литералы, кроме байтового, позволяют использовать суффиксы,
такие как 57u8 и _ в качестве разделителя, например 1_000
Числовые литералы Пример
десятичный			98_222
шестнадцатиричный	0xff
восьмиричный		0o77
бинарный			0b1111_0000
байтовый			b'A'

f32 и f64 два типа для чисел с плавающей точкой

Базовые математические операции для всех числовых типов:
1. сложение
2. вычитание
3. умножение
4. деление
5. получение остатка

bool занимает 1 байт и имеет два значения: true и false

char определяется с помощью одинарных кавычек и имеет размер 4 байта

Сложные типы данных:
1. кортежи
2. массивы

Кортеж тип включающий в себя несколько разные типы
Кортежи после объявления не могут расти или уменьшаться в размере
let tup: (i32, f64, u8) = (500, 6.4, 1);

Массивы состоят из элементов одного типа и тоже имеют фиксированную длинну
let a = [1, 2, 3, 4, 5];
let a = [i32; 5] = [1, 2, 3, 4, 5];
let a = [3; 5];
Mожно получить доступ к элементу массива используя индекс

Functions
Функции объявляются при помощи ключевого слова fn и круглых скобок
фигурные скобки указывают компилятору тело функции
для имён функции используется snake case
функция может быть объявлена в любом месте
при наличии параметров для каждого из них должен быть объявлен тип через запятую
Операторы - это инструкции, которые выполняют какие-либо действия и не возвращают значение
Выражение вычисляют результируещее выражение
